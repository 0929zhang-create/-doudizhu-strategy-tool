# 斗地主策略助手（在线版）

这是一个基于网页的斗地主辅助策略工具，支持：

- 选择你的身份（地主或农民）  
- 通过点击选择手牌  
- 选择上家和下家的身份与出牌  
- 自动分析牌型（单牌、对牌、三张、炸弹、顺子）  
- 智能推荐出牌策略，支持不出建议  
- 支持对局历史输入辅助分析  

## 使用说明

- 打开网页后选择你的身份  
- 点击选择你的手牌  
- 点击选择上家和下家的身份及出牌  
- 填写当前对局历史（可选）  
- 系统自动实时推荐最优出牌策略
